# PyMLB
Using MLB API to extract pitchfx data and plot them for visualization

### How to:
After downloading `PyMLB.py` in your work environment:

```python
import PyMLB
```

### Available Function Calls:

```python
get_attendance()
get_division()
lookup_division()
get_linescore()
get_pitchfx()
lookup_player()
lookup_player_info()
player_game_stat()
get_schedule()
get_season()
get_standings()
lookup_team()
lookup_roster()
```
